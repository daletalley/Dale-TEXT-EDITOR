# TEXT-EDITOR 
## Overview
- Simple, offline-capable text editor.
- Built as a Progressive Web App (PWA).
- Uses IndexedDB, webpack, and service workers.

## Contents
- Installation
- How to Use
- Screenshots
- License Info


## Installation
1. Install Node.js.
2. In terminal, navigate to app's root:
3. Install dependencies:
npm install

## How to Use
1. Start the app:
npm run start
2. Access via `localhost:3000`.
